"Магазин Звучёк" — это система управления музыкальным магазином с графическим интерфейсом на Python + Tkinter. Программа предназначена для автоматизации работы музыкального магазина, включая управление товарами, продажи, возвраты и поиск.

Основные возможности:

1.Управление товарами:

	Просмотр каталога музыкальных инструментов (8 предустановленных товаров)

	Добавление новых товаров на склад

	Удаление товаров со склада

	Просмотр информации о товарах: название, бренд, цена, количество

2.Фильтрация по жанрам:

	Возможность фильтровать инструменты по музыкальным жанрам

	Поддерживаемые жанры: рок, джаз, классика, поп, электроника

	Функция "Все" для отображения всех товаров

3.Система продаж:

	Создание новых продаж с выбором продавца и покупателя

	Добавление товаров в корзину продажи с указанием количества

	Удаление товаров из корзины

	Автоматический расчет суммы продажи

	Завершение продажи с формированием чека

4.Система возвратов:

	Создание возвратов товаров

	Указание причины возврата

	Отметка о наличии/отсутствии чека

	Просмотр истории всех созданных возвратов

	Подробная информация о каждом возврате

5.Поисковая система:

	Поиск товаров по стилю (жанру)

	Поиск товаров по бренду

	Возможность комбинированного поиска

	Пользовательский интерфейс:

	6 вкладок для удобной навигации:

		Жанры (фильтрация товаров)

	 	Склад (управление товарами)

		Продажа (оформление продаж)

		Создать возврат

 		Список возвратов

	 	Поиск

Технические особенности:

Архитектура:

	Классы: Seller, Customer, MusicalInstrument, Sale, Return, Database, MusicShop

	Четкое разделение бизнес-логики и интерфейса

	Объектно-ориентированный дизайн

Функционал продаж:

	Автоматическое обновление количества товаров на складе

	Проверка наличия товара перед добавлением в продажу

	Поддержка добавления одного товара несколько раз (увеличение количества)

	Возможность частичного удаления товаров из продажи

Работа с данными:

Предустановленные тестовые данные:

	4 продавца

	5 покупателей

	8 музыкальных инструментов разных категорий

	Автоматическая генерация ID для новых записей

	Хранение истории продаж и возвратов

Требования к системе:

	Python 3.6 или выше

	Стандартная библиотека Python (без дополнительных зависимостей)

	Операционная система: Windows, Linux, macOS
