Магазин "Мелодия" — Система управления музыкальным магазином
Описание программы:
«Магазин Мелодия» — полнофункциональная музыкальная система розничными торговыми инструментами с графическим интерфейсом на Python + Tkinter . Комплексное решение для автоматизации делового музыкального магазина.
​

 Основной функционал
1. Управление товарным ассортиментом
 Просмотр каталога (8 инструментов: гитары, синтезаторы и др.)

 Фильтрация по жанрам (Рок, Поп, Джаз, Электронная, Классика...)

 Подробная информация о товаре (написание, цена, жанры)

 Цены в рублях с автоформатированием тысяч

2. Работа с корзиной покупками
 Множественный выбор товара (Ctrl+клик/Shift+диапазон)

 Добавление выделенных товаров в корзину

 Удаление выбранных товаров из корзины

 Полная очистка корзины (с подтверждением)

 Автоматический расчет итоговой суммы

3. Оформление заказов
 Формирование чека с:

  1.Дата и время ( 2026-01-20 20:25)

  2.Списком товаров и цен

  3.Итоговой суммы

  4.Подтверждение заказа в отдельном окне
4.Технические особенности

  Python 3.6 или выше
 
  Стандартная библиотека Python (Tkinter, datetime)
 
  Операционная система: Windows, Linux, macOS
